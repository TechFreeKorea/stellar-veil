STELLAR VEIL

온라인 2인 우주 관측 추리 게임

게임 진행

  • 관측소를 만들고 코드를 공유하거나, 코드로 참가하세요.
  • 각자 천체 5개를 배치 → 둘 다 완료되면 관측 시작!
  • 상대 성도에 광자 빔을 쏘아 천체를 추리합니다.

천체 종류 (5개, 4색)

  • 백색왜성: 큰 삼각형(6칸) × 2개
  • 적색거성 평행사변형(3칸), 황색왜성 삼각형(3칸), 청색초거성 마름모(4칸) = 각 1개

배치 규칙

  • 백색왜성: 45도 회전 / 황색왜성·적색거성: 90도 회전 / 청색초거성: 회전 없음
  • 뒤집기 불가
  • 서로 다른 천체는 변끼리 맞닿을 수 없음 (꼭짓점은 OK)
  • 성도 밖으로 나가거나 겹칠 수 없음

광자 빔 반사

  • 가장자리 라벨 클릭 → 직선 이동
  • 대각선 천체 → 중력렌즈 효과로 90도 굴절
  • 평면 천체 → 중력 반사로 역방향 반사 (180도)

스펙트럼 분석

  • 적+청=자외선, 적+황=적외선, 청+황=오로라
  • 적+청+황=블랙홀 (광자 흡수, 출구 정보 없음)
  • 백색왜성은 굴절만, 스펙트럼 영향 없음
